## 💡 How to Use

### Basic Workflow

1. **Enter Property Details**
   - Address, price, beds, baths, sqft, year built, DOM
   - Choose contact type (Owner/Realtor)
   - Select property type (House/Land)

2. **Add Financial Data**
   - **For Houses:** Enter 3 comps, monthly rent, repairs
   - **For Land:** Builder price per ¼ acre, lot size

3. **Review Analysis**
   - AI Deal Score (0-100)
   - Verdict (GO/MAYBE/NO GO)
   - Key numbers (ARV, MAO 70%, MAO RBP 88%)
   - Deal strengths & killers

4. **Select Strategy**
   - Choose from 4 proven strategies
   - Get full word-for-word scripts
   - See calculated metrics

5. **Generate Deliverables**
   - Download Deliverable A (Analysis)
   - Download Deliverable B (Offer Package)
   - Or download both at once

6. **Make the Call**
   - Switch to Call Mode tab
   - Use provided scripts
   - Add notes and schedule follow-ups

### Advanced Features

- **Dark Mode:** Toggle in top navigation bar
- **Save Deals:** Save unlimited deals for comparison
- **Print:** Print/PDF current analysis
- **Templates:** Access pre-built documents
- **CRM:** Track multiple deals simultaneously

## 🎓 Strategy Guide

### When to Use Each Strategy

**Cash Wholesale**
- Use when: Price ≤ 70% ARV
- Best for: Quick flips to investors
- Timeline: 7-14 days
- Your profit: Assignment fee ($5-15K)

**Subject-To**
- Use when: Low interest rate (<6%), has mortgage balance
- Best for: Cash flow properties
- Timeline: 14-30 days
- Your profit: Monthly cash flow + equity

**Retail Buyer Program (RBP)**
- Use when: Price ≤ 88% ARV
- Best for: Properties needing light rehab
- Timeline: 14-21 days
- Your profit: Assignment fee to retail buyer

**Builder Assignment**
- Use when: Land deal with builder network
- Best for: Buildable lots
- Timeline: 21-30 days
- Your profit: 15-25% spread
